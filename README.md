# Hamster Script

Linux 服务器管理脚本，dialog 菜单界面。

## 安装

```bash
bash <(curl -sL https://github.com/3106961196/hamster-script/raw/main/setup.sh)
```

或：

```bash
git clone https://github.com/3106961196/hamster-script.git
cd hamster-script
sudo ./setup.sh
```

## 使用

```bash
cs              # 主菜单
cs update       # 更新
nt [QQ] [端口]  # NapCat
hamster-tmux    # tmux 桌面
```
